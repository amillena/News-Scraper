# Travel News-Scraper

## Project Description

Travel News Scraper is an app that allows user to leave comments on the latest travel news artile from NY Times. Mongoose and Cheerio were used to scraped New York Times for articles.

## Technologies Used

* HTML
* CSS
* JavaScript
* jQuery
* Bootstrap
* Node.js
* Express
* MongoDB
* Mongoose
* Cheerio
